**現在のリリース:** v0.1  
**ステータス:** ワーキングペーパー（継続研究）

本リポジトリは、人工知能（AI）を「現代の錬金術」―すなわち変換のプロセスとして研究するプロジェクトです。

古典的な錬金術が賢者の石や物質の変成を追究したように、本研究は知性の変換、経済構造の再編、自律システムに対する人間の統制の限界を探究します。

---

## 研究プログラム — Modern Alchemy (2026)

本研究は人工知能の変革的影響を以下の3領域から分析します：

1. **Digital Philosopher’s Stone** — 知性の変換と自律性  
2. **Economic Transmutation** — 経済構造の再編  
3. **Stone’s Shadow** — リスクと統制の限界
